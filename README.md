# EarPhone Music 🎧

**耳に当てるだけで音楽を検索できる、革新的な音楽検索アプリ**

## 📱 アプリの概要

EarPhone Musicは、近接センサーを活用した新しい音楽体験を提供するiOSアプリです。
SpotifyとYouTubeの音楽を、シンプルで直感的なインターフェースで検索できます。

## ✨ コンセプト

### 1. **耳フォン（EarPhone）体験**
- 📱 **耳に当てると自動でレシーバーモードに切り替え**
- 👂 近接センサーで耳の接近を検知
- 🔊 耳を離すとスピーカーモードに自動復帰

### 2. **検索特化型デザイン**
- 🔍 SpotifyとYouTubeの音楽検索に特化
- 🌐 Webブラウザ内で検索結果を表示
- 📱 アプリに強制的に飛ばないスマートな設計

### 3. **ミニマルUI**
- 🎨 言葉がなくても使い方がわかる直感的なデザイン
- ⬅️➡️ スワイプでSpotify/YouTubeを簡単切り替え
- 🎯 余計な機能を排除したシンプルな操作性

## 🎨 デザインコンセプト

### Spotifyページ
- **背景色**: 黒 (#191414) - Spotify公式カラー
- **アクセント**: 緑 (#1DB954) - Spotifyブランドカラー
- **雰囲気**: モダン、クール、音楽に没入できる暗めの空間

### YouTubeページ
- **背景色**: 白 (#FFFFFF) - クリーンで明るい
- **アクセント**: 赤 (#FF0000) - YouTube公式カラー
- **雰囲気**: 明るく、親しみやすい、動画コンテンツに適した空間

### 共通デザイン要素
- 🔵 大きなアイコン（120x120）で視認性抜群
- 🔍 丸みを帯びた検索バー（角丸50）
- 👂 近接センサーのリアルタイム表示
- ・ページインジケーターでナビゲーション

## 🎯 ターゲットユーザー

- **音楽好き**: SpotifyとYouTubeを頻繁に使う人
- **シンプル志向**: 複雑な機能より直感的な操作を好む人
- **プライバシー重視**: アプリに飛ばされるのを嫌う人
- **イヤホン使用者**: 耳にスマホを当てて通話するように音楽を楽しみたい人

## 🚀 主な機能

### 1. デュアル検索システム
- **Spotify**: 音楽ストリーミングの王道
- **YouTube**: 世界最大の動画プラットフォーム
- スワイプ1つで瞬時に切り替え

### 2. インテリジェントな近接センサー
- 耳に近づくとレシーバーモードに**最速切り替え**
- リアルタイムで状態を視覚的に表示
- 自動的に最適な音声出力を選択

### 3. アプリ内Webブラウザ
- WKWebViewを使用した高速ブラウジング
- 戻る・進む・リロード機能完備
- アプリへの強制遷移をブロック

### 4. キーボード操作
- 画面タップでキーボードを閉じる
- Enter キーで即座に検索実行

## 🎨 アイコンデザインの指針

### アイコンに含めるべき要素：
1. **耳のシンボル** 👂
   - アプリのコアコンセプト「耳に当てる」を表現

2. **音楽要素** 🎵
   - 音符、ヘッドホン、イヤホンなどの音楽的なシンボル

3. **カラースキーム**
   - **メインカラー**: Spotify緑 (#1DB954) または YouTube赤 (#FF0000)
   - **サブカラー**: 黒 (#191414) または 白 (#FFFFFF)
   - グラデーション効果で両方のサービスを統合

4. **デザインスタイル**
   - **モダン**: iOS 17の最新デザイントレンドに準拠
   - **ミニマル**: シンプルで認識しやすい
   - **丸みのある**: 親しみやすく優しい印象
   - **グラデーション**: 音楽の流れを表現

### アイコンデザインの例：
```
🎨 提案1: 耳+音符
- 耳のシルエットに音符が流れ込む様子
- グラデーション: 緑→赤（Spotify→YouTube）

🎨 提案2: イヤホン
- イヤホンのシンプルなアイコン
- 背景に音波のグラデーション

🎨 提案3: 耳+サーチアイコン
- 耳のシルエット + 虫眼鏡
- 「検索」のコンセプトを明確に表現
```

## 💡 アプリの独自性

1. **近接センサーの活用**: 他の音楽アプリにはない独自機能
2. **検索特化**: 再生ではなく「発見」に焦点
3. **プラットフォーム横断**: SpotifyとYouTubeを1つのUIで
4. **プライバシー保護**: アプリに飛ばないブラウザ内検索

## 🔧 技術スタック

- **言語**: Swift
- **フレームワーク**: SwiftUI
- **最小バージョン**: iOS 17.5
- **特殊機能**:
  - AVAudioSession (近接センサー)
  - WKWebView (アプリ内ブラウザ)
  - TabView (ページング)

## 🎯 アプリ名の意味

**EarPhone Music** = Ear + Phone + Music
- **Ear**: 耳に当てる独自の体験
- **Phone**: スマートフォンの近接センサーを活用
- **Music**: 音楽検索の楽しさ
